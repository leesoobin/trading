# 전략 개요 (Strategy Overview)

> **핵심 철학**
> 횡보(Range) 구간이 확정된 후, 박스 상단/하단의 유동성이 스윕되는 순간을 포착하여 **반대 방향으로 진입**한다. 횡보 구간은 Bigboy가 주문을 축적하는 Range 단계이며, 상단/하단의 Stop Loss를 터뜨리는 False Breakout이 최고의 진입 기회다.

## 전략 성격
| 항목 | 값 |
|------|-----|
| 유형 | 횡보 매매 / 레인지 트레이딩 |
| 목표 승률 | 50~55% |
| 목표 RR | 1:1 ~ 1:2 |
| 매매 빈도 | 높음 |
| 난이도 | 중간 |
| 적합 장세 | **횡보장** (박스권 확정 후) |

## 전략의 전제 조건
- **횡보가 확정**되어야 함 (최소 스윙 포인트 3개 이상 형성)
- 추세장에서는 이 전략을 **사용하지 않음** → 전략 1(추세추종) 또는 전략 2(스윕 헌팅)로 전환
- 박스 상단/하단은 **정확한 수평선이 아닌 "구간(Zone)"으로** 설정 (스탑헌팅 대비)
- 양방향 매매 가능하나, **HTF 대추세 방향의 포지션에 더 큰 비중**

# 1. 핵심 개념 정의

## 1-1. 횡보(Range) 구간 식별

### 횡보 확정 조건
1. 가격이 **일정 범위 내에서** 상하로 진동
2. **스윙 포인트가 최소 3개** 이상 형성 (고점 2개 + 저점 1개, 또는 그 반대)
3. 이평선들이 **수렴/교차** 상태 (정배열도 역배열도 아닌 상태)
4. HTF에서 **BOS가 발생하지 않음** (구조가 깨지지 않음)

### 횡보 구간 그리기
```
박스 상단 Zone ─────────────── ← EQH / 직전 고점들 (유동성 축적)
   │                          │
   │     Balanced Zone        │ ← 균형가 (박스 중앙)
   │     (Equilibrium)        │
   │                          │
박스 하단 Zone ─────────────── ← EQL / 직전 저점들 (유동성 축적)
```

### 주의사항
- 박스를 **수평선 1개가 아닌 Zone(구간)으로** 설정
  - 상단 Zone: 최고점 ~ 최고점 아래 일정 범위
  - 하단 Zone: 최저점 ~ 최저점 위 일정 범위
- 이유: 스마트머니는 박스 경계를 **살짝 넘어서** 유동성을 스윕함

## 1-2. Balanced Price Zone (균형가)

- 박스의 **정중앙** 가격대
- 1차 익절 목표로 활용 (RR 1:1 달성 지점)
- 균형가를 기준으로:
  - 박스 상단 Zone에 있으면 → **숏 기회 탐색**
  - 박스 하단 Zone에 있으면 → **롱 기회 탐색**

## 1-3. 박스 내 유동성 구조

```
박스 상단 ── EQH ── 매수자 Stop(숏 청산) + 매수 돌파 주문 = 상방 유동성
    │
    │  IDM (Inducement) ← 박스 내부 스윙 포인트
    │  트레이더들이 매매하고 싶어하는 미끼 구간
    │
박스 하단 ── EQL ── 매도자 Stop(롱 청산) + 매도 돌파 주문 = 하방 유동성
```

- 박스 상단/하단에는 **양쪽 방향의 Stop Loss가 축적**
- Bigboy는 한쪽 유동성을 스윕 → 반대 방향으로 이동 → 다른 쪽 유동성도 스윕
- → **"외부 유동성을 터뜨리고, 내부 유동성을 터뜨린다"**

## 1-4. False Breakout (페이크 아웃)

### 정의
- 가격이 박스 상단/하단을 **일시적으로 돌파한 후** 다시 박스 안으로 복귀하는 현상
- = 유동성 스윕 + 박스 복귀

### False Breakout 확인 방법
1. 가격이 박스 경계를 **캔들 꼬리로** 넘김 (몸통은 안쪽 마감)
2. 또는 **1~2 캔들** 몸통으로 넘었다가 빠르게 복귀
3. 후속 캔들이 **반전 PA**를 형성 (핀바/잉걸핑/인사이드바 돌파)
4. 진짜 돌파와의 구별: 3캔들 이상 박스 바깥에서 마감하면 진짜 돌파 가능성 높음

### False Breakout vs 진짜 Breakout
| 구분 | False Breakout (이 전략 적용) | 진짜 Breakout (전략 전환) |
|------|---------------------------|----------------------|
| 캔들 마감 | 꼬리만 넘기거나 빠르게 복귀 | 몸통으로 넘겨 3캔들 이상 유지 |
| 후속 움직임 | 즉각 반전 | 돌파 방향으로 추가 진행 |
| 거래량 | 스윕 시 스파이크 후 감소 | 돌파 후에도 거래량 유지 |
| 대응 | 반대 방향 진입 | 전략 1/2로 전환 |

# 2. 멀티타임프레임 (MTF) 분석 체계

## 2-1. 타임프레임 역할 분담

| 타임프레임 | 역할 | 확인 사항 |
|-----------|------|----------|
| **일봉/4시간봉 (HTF)** | 대추세 방향 + 횡보 확인 | 횡보 확정, 박스 범위 설정, 대추세 방향 |
| **1시간봉 (MTF)** | 박스 내 구조 관찰 | 박스 상/하단 접근, 내부 스윙 구조 |
| **15분/5분봉 (LTF)** | 정밀 진입 | False Breakout 감지, PA 확인, 진입 |

## 2-2. 분석 플로우

```
[Step 1] HTF(4H) 횡보 확정
    │
    ├── 스윙 포인트 3개 이상 → 횡보 확정
    ├── 박스 상단/하단 Zone 설정
    ├── 대추세(일봉) 방향 확인 → 유리한 방향 결정
    └── 이평선 수렴/교차 확인
    │
[Step 2] 대추세 방향에 따른 편향(Bias) 설정
    │
    ├── 일봉 상승 추세 → 박스 하단에서 롱 우선 (비중 ↑)
    ├── 일봉 하락 추세 → 박스 상단에서 숏 우선 (비중 ↑)
    └── 일봉 횡보 → 양방향 동일 비중
    │
[Step 3] 박스 상/하단 접근 감시 (MTF)
    │
    ├── 가격이 박스 상단 Zone 진입 → 숏 준비
    ├── 가격이 박스 하단 Zone 진입 → 롱 준비
    └── 가격이 균형가 부근 → 대기
    │
[Step 4] LTF(5m/15m) False Breakout + PA 확인
    │
    ├── 박스 경계 스윕 캔들 발생
    ├── 반전 PA 확인 (핀바/잉걸핑/인사이드바)
    └── 진입 실행
```

# 3. 진입/손절/익절 규칙

## 3-1. 롱 진입 (박스 하단 스윕 후)

### 진입 조건 체크리스트
- [ ] HTF(4H): 횡보 확정 (스윙 포인트 3개 이상)
- [ ] HTF(4H): 박스 하단 Zone에 유동성 축적 확인 (EQL 또는 반복된 저점)
- [ ] 가격이 박스 하단 Zone에 진입 또는 하방 스윕 발생
- [ ] LTF(5m/15m): False Breakout 캔들 확인 (긴 아래꼬리 + 박스 안쪽 마감)
- [ ] LTF(5m/15m): Bullish PA 출현 (Hammer, Bullish Engulfing, 인사이드바 상단 돌파)
- [ ] (가산) 박스 하단이 HTF Demand Zone 또는 OB와 겹침
- [ ] (가산) 대추세(일봉) 방향이 상방

### 진입 시점
- **기본**: False Breakout 후 Bullish PA 캔들 종가 확정 시
- **공격적**: 박스 하단 Zone 진입 시 지정가(Limit) 배치
- **인사이드바 활용**: 인사이드바 발생 → 어미봉 상단 돌파 시 진입 (어미봉 하단 손절)

### 손절
- **기본**: 박스 하단 Zone의 극저점 하방 (= False Breakout 캔들 꼬리 끝 하방)
- 이 전략에서 손절은 비교적 짧음 (박스 Zone 범위 + buffer)
- ⚠️ 진짜 하방 돌파가 발생하면 즉시 손절 (구조 이탈)

### 익절
- **1차 익절 (50%)**: 균형가(박스 중앙) — RR 약 1:1
- **2차 익절 (50%)**: 박스 상단 Zone 하단 — RR 약 1:2
- 또는 **가산 조건 충족 시 (대추세 정렬)**: 잔여분 트레일링 → 박스 상방 돌파까지 보유

## 3-2. 숏 진입 (박스 상단 스윕 후)

### 진입 조건 체크리스트
- [ ] HTF(4H): 횡보 확정 (스윙 포인트 3개 이상)
- [ ] HTF(4H): 박스 상단 Zone에 유동성 축적 확인 (EQH 또는 반복된 고점)
- [ ] 가격이 박스 상단 Zone에 진입 또는 상방 스윕 발생
- [ ] LTF(5m/15m): False Breakout 캔들 확인 (긴 위꼬리 + 박스 안쪽 마감)
- [ ] LTF(5m/15m): Bearish PA 출현 (Shooting Star, Bearish Engulfing, 인사이드바 하단 돌파)
- [ ] (가산) 박스 상단이 HTF Supply Zone 또는 OB와 겹침
- [ ] (가산) 대추세(일봉) 방향이 하방

### 진입 시점
- **기본**: False Breakout 후 Bearish PA 캔들 종가 확정 시
- **공격적**: 박스 상단 Zone 진입 시 지정가(Limit) 배치
- **인사이드바 활용**: 인사이드바 발생 → 어미봉 하단 돌파 시 진입

### 손절
- **기본**: 박스 상단 Zone의 극고점 상방 (= False Breakout 캔들 꼬리 끝 상방)

### 익절
- **1차 익절 (50%)**: 균형가(박스 중앙) — RR 약 1:1
- **2차 익절 (50%)**: 박스 하단 Zone 상단 — RR 약 1:2
- 또는 **가산 조건 충족 시 (대추세 정렬)**: 잔여분 트레일링 → 박스 하방 돌파까지 보유

# 4. 매매 보류 조건 (No Trade Zone)

다음 조건 중 하나라도 해당하면 **매매하지 않음**:

1. **횡보 미확정** — 스윙 포인트 3개 미만, 아직 박스가 형성되지 않음
2. **가격이 균형가(박스 중앙) 부근** — 상단/하단 어느 쪽도 아닌 애매한 위치
3. **박스 상/하단이 이미 3회 이상 터치됨** — Zone 붕괴 가능성 높음 (Freshness 저하)
4. **박스 범위가 너무 좁음** — 의미있는 RR 확보 불가
5. **박스 범위가 너무 넓음** — 손절이 지나치게 넓어짐
6. **진짜 Breakout 발생** — 몸통으로 3캔들 이상 박스 밖 마감 → 전략 1/2로 전환
7. **주요 경제 이벤트 직전** — 횡보를 깨는 촉매가 될 수 있음
8. **RR 1:1 미만** — 최소 균형가까지의 RR도 확보 안 됨

# 5. 시나리오 상세 (실전 플로우)

```
[전제] 4H 횡보 확정, 스윙 포인트 3개 이상, 대추세 상방
[발생] 가격이 박스 하단 EQL을 하방 스윕 (긴 아래꼬리)
[확인] LTF(5m) Hammer 또는 Bullish Engulfing 출현

[익절] 1차 균형가 (50%), 2차 박스 상단 (50%)
[대추세 정렬] 대추세 상방이므로 2차 익절분 중 일부 트레일링 가능

### 시나리오 B: 인사이드바 False Breakout 전환

[전제] 4H 횡보 확정, 박스 하단에 인사이드바 형성
[발생] 인사이드바가 하방 돌파하는 척 → 박스 안으로 복귀(False Breakout)
[진입] 어미봉 상단 돌파 시 롱 진입
[손절] 어미봉 하단 하방 (타이트한 손절)
[익절] 1차 균형가, 2차 박스 상단
[장점] 손절이 매우 짧아 높은 RR 가능
```

# 6. 횡보 탈출(Breakout) 전환 시나리오

횡보는 영원히 지속되지 않으며, 결국 한 방향으로 돌파한다. **횡보 매매 중 진짜 Breakout이 발생하면**:

### 전환 조건
1. 캔들 몸통으로 박스 상/하단을 완전히 돌파
2. 3캔들 이상 박스 바깥에서 마감 유지
3. 돌파 방향으로 BOS 발생

### 대응
- **현재 포지션이 돌파 방향과 반대**인 경우 → 즉시 손절
- **현재 포지션이 돌파 방향과 동일**인 경우 → 트레일링 스탑으로 수익 극대화
- 이후 매매 → 전략 1(OB 눌림목) 또는 전략 2(스윕 헌팅)로 전환
- ⚠️ 돌파 직후 "따라가기"는 위험 → 되돌림을 기다려 OB에서 진입

### 대추세 방향 Breakout 시 추가 수익
```
[상황] 대추세 상방 + 횡보 상방 돌파
[대응] 박스 하단에서 잡은 롱 포지션의 2차 익절분을 트레일링
[근거] 횡보 길이와 돌파 후 추세 강도는 정비례 (매물대 원리)
[익절] BOS마다 손절을 올리며 추세 라이딩
```

# 7. 레버리지 및 포지션 관리 특이사항

횡보 매매는 다른 전략과 **포지션 관리 방식이 다름**:

| 항목 | 추세추종/스윕 전략 | 횡보 매매 (이 전략) |
|------|-----------------|------------------|
| 레버리지 | 가능 (추세 방향) | **낮게 또는 미사용 권장** |
| 포지션 모으기 | 어려움 | 가능 (박스 하단에서 분할 매수) |
| 홀딩 기간 | 길 수 있음 | 짧음 (수 시간 ~ 수 일) |
| 매매 빈도 | 낮음~중간 | 높음 |
| 심리적 부담 | 보통 | 피로도 높음 (빈번한 매매) |

### 횡보 구간에서 포지션 모으기 (선택적)
- 대추세 방향과 일치하는 포지션을 **박스 하단(롱) 또는 상단(숏)에서 분할 매수/매도**
- 손절 안 걸리면 유지, 균형가 도달 시 80% 익절
- 잔여 20%는 Breakout 대비 보유
- 레버리지 **미사용** 권장 (포지션을 모아가기 어려우므로)

# 8. AI 전문가 에이전트 디렉션

## 에이전트 역할 정의
> 이 에이전트는 "SMC 횡보 박스권 전문가"로서, 횡보 확정 → 박스 설정 → 유동성 스윕 감지 → PA 확인의 순서로 분석하고 매매 판단을 제공한다.

## 분석 파이프라인 (순서대로 실행)

### Phase 1: 횡보 확정 (HTF)
1. 4H 차트에서 스윙 포인트 개수 확인 (3개 이상 필요)
2. BOS 미발생 확인 (구조가 깨지지 않음)
3. 이평선 수렴/교차 상태 확인
4. → **횡보 미확정 시 "전략 무효" 판정, 다른 전략으로 전환 권고**

### Phase 2: 박스 설정 및 대추세 확인
5. 박스 상단/하단 Zone 설정
6. 균형가(중앙) 계산
7. 대추세(일봉) 방향 확인 → Bias 설정
8. 박스 상/하단의 유동성 축적 상태 확인 (EQH/EQL)

### Phase 3: 가격 위치 판단 (MTF)
9. 현재 가격이 박스 어디에 위치하는지 판단
   - 상단 Zone → 숏 준비
   - 하단 Zone → 롱 준비
   - 균형가 부근 → 대기
10. 박스 상/하단 터치 횟수 확인 (3회 이상이면 경고)

### Phase 4: False Breakout + PA 확인 (LTF)
11. 박스 경계 스윕/False Breakout 발생 여부
12. 반전 PA 캔들 탐색 (핀바/잉걸핑/인사이드바)
13. 진입 가격, 손절 가격, 익절 목표가 산출
14. RR 계산 → 1:1 미만이면 패스

### Phase 5: 리스크 관리
15. 포지션 사이즈 산출 (계좌 1~2%)
16. 대추세 비정렬 시 사이즈 50% 축소
17. 분할 익절 계획 (50% 균형가 / 50% 반대편 Zone)

### Phase 6: 최종 판단 출력
```
[판정] 롱/숏/보류
[신뢰도] 상/중/하
[장세] 횡보 확정 / 미확정
[대추세 방향] 상방/하방/횡보
[가격 위치] 상단 Zone / 하단 Zone / 중앙
[스윕/False Breakout] 확인/미확인
[PA 확인] 핀바/잉걸핑/인사이드바/미확인
[진입가] xxxx
[손절가] xxxx (박스 Zone 극단점 기준)
[익절 1차] xxxx (균형가, R:R 1:x)
[익절 2차] xxxx (반대편 Zone, R:R 1:x)
[포지션 사이즈] 계좌의 x%
[주의사항] (해당 시)
```

# 9. 전략 한계 및 주의사항

1. **횡보 판단 오류**: 횡보로 보였으나 실은 추세의 일부(Flag/Pennant)일 수 있음
2. **진짜 Breakout 리스크**: False Breakout으로 판단했으나 진짜 돌파인 경우 → 손절 필수
3. **낮은 RR**: 추세추종/스윕 전략 대비 RR이 낮아, 승률이 받쳐줘야 함
4. **피로도**: 매매 빈도가 높아 심리적 에너지 소모가 큼
5. **박스 붕괴 3회 규칙**: 같은 방향으로 3회 이상 터치된 Zone은 깨질 가능성 높음
6. **뉴스 Breakout**: 경제 이벤트가 횡보를 강제로 깨뜨릴 수 있음

# 부록: 용어 정리

| 용어 | 설명 |
|------|------|
| Range | 횡보 구간, 가격이 일정 범위 내에서 진동 |
| Balanced Price Zone | 균형가, 박스의 정중앙 가격대 |
| EQH | Equal Highs — 비슷한 높이의 고점이 반복 형성 |
| EQL | Equal Lows — 비슷한 높이의 저점이 반복 형성 |
| False Breakout | 박스 경계를 일시적으로 돌파 후 복귀하는 현상 |
| Breakout | 박스를 진짜로 돌파하여 새로운 추세 시작 |
| Inside Bar | 이전 캔들(어미봉) 범위 안에 형성된 캔들(새끼봉) |
| Bias | 대추세 방향에 따른 매매 편향 |
| Accumulation | 횡보 중 기관이 포지션을 축적하는 과정 |
