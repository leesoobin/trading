# SMC 전략 비교 매트릭스

> 4가지 SMC 전략 + 이평선 전략을 통합 비교하여, **어떤 장세에서 어떤 전략을 사용할지** 빠르게 판단하기 위한 매트릭스이다.

---

|------|-----------------|----------------|-----------------|-----------------|------------|
| **유형** | 추세추종 | 추세 전환/역추세 | 횡보 매매 | 인디케이터 기반 | 추세추종 |
| **목표 승률** | 50~60% | 30~40% | 50~55% | 55~65% | 50~60% |
| **목표 RR** | 1:2~1:3 | 1:3~1:5+ | 1:1~1:2 | 1:2~1:3 | 1:2~1:3 |
| **매매 빈도** | 중간 | 낮음 | 높음 | 낮음 | 중간 |

## 2. 장세별 전략 선택 가이드

### 판단 플로우차트
```
현재 시장 상태는?
    │
    ├── [추세 확립] BOS 반복, 이평선 정/역배열
    │       │
    │       ├── OB 접근 중 → 전략 1 (OB 눌림목)
    │       ├── Golden OB 시그널 → 전략 4 (Golden OB)
    │       └── 이평선 눌림목 → 이평선 전략
    │
    ├── [추세 전환 의심] 유동성 풀 도달, 스윕 발생
    │       │
    │       ├── CHOCH + BOS 확정 → 전략 2 (스윕 헌팅)
    │       └── CHOCH만, BOS 미확정 → 대기
    │
    ├── [횡보 확정] 스윙 포인트 3개+, BOS 없음
    │       │
    │       ├── 박스 상/하단 접근 → 전략 3 (박스권 매매)
    │       └── 균형가 부근 → 대기
    │
    └── [불명확] 어디에도 해당하지 않음
            │
            └── 매매 보류 (No Trade)
```

### 장세별 매트릭스

| 장세 | 1순위 전략 | 2순위 전략 | 사용 불가 |
|------|----------|----------|----------|
| **강한 상승 추세** | 전략 1 (OB 눌림목 롱) | 전략 4 (Golden OB 롱) | 전략 2 (숏 방향) |
| **강한 하락 추세** | 전략 1 (OB 눌림목 숏) | 전략 4 (Golden OB 숏) | 전략 2 (롱 방향) |
| **상승 추세 고점 도달** | 전략 2 (스윕 숏 준비) | 전략 1 (마지막 눌림목) | 전략 3 |
| **하락 추세 저점 도달** | 전략 2 (스윕 롱 준비) | 전략 1 (마지막 눌림목) | 전략 3 |
| **횡보 / 박스권** | 전략 3 (박스권 매매) | - | 전략 1, 전략 4 |
| **횡보 → 돌파 직후** | 전략 1 (돌파 후 눌림목) | 전략 4 | 전략 3 |
| **급변동 / 이벤트** | 전체 보류 | - | 전체 |

## 4. 기대값(Expected Value) 비교

> 기대값 = (승률 × 평균 수익) - ((1-승률) × 평균 손실)
> 100회 매매 기준, 매회 1% 리스크 가정

### 전략 1: OB 눌림목
```
승률 55%, RR 1:2.5
기대값 = (0.55 × 2.5%) - (0.45 × 1%) = 1.375% - 0.45% = +0.925%/매매
100회 기대 수익: +92.5%
```

### 전략 2: 스윕 헌팅
```
승률 35%, RR 1:4
기대값 = (0.35 × 4%) - (0.65 × 1%) = 1.40% - 0.65% = +0.75%/매매
100회 기대 수익: +75%
빈도가 낮으므로 100회 달성에 오랜 기간 필요
```

### 전략 3: 박스권 매매
```
승률 52%, RR 1:1.5
기대값 = (0.52 × 1.5%) - (0.48 × 1%) = 0.78% - 0.48% = +0.30%/매매
100회 기대 수익: +30%
빈도가 높으므로 회전율로 보상
```

### 전략 4: Golden OB
```
승률 60%, RR 1:2.5
기대값 = (0.60 × 2.5%) - (0.40 × 1%) = 1.50% - 0.40% = +1.10%/매매
100회 기대 수익: +110%
빈도가 낮으므로 100회 달성에 오랜 기간 필요
```

### 기대값 매트릭스 요약
| 전략 | 매매당 기대값 | 빈도 | 월간 예상 기대값 (20회 매매 기준) |
|------|------------|------|--------------------------|
| 전략 1 | +0.925% | 중간 (~15~20회/월) | +13.9~18.5% |
| 전략 2 | +0.75% | 낮음 (~5~8회/월) | +3.75~6.0% |
| 전략 3 | +0.30% | 높음 (~20~30회/월) | +6.0~9.0% |
| 전략 4 | +1.10% | 낮음 (~5~10회/월) | +5.5~11.0% |

> ⚠️ 위 수치는 이론적 추정치이며, 실제 성과는 시장 환경, 실행력, 심리 관리에 따라 크게 달라진다. 반드시 백테스트와 실매매 기록으로 검증해야 한다.

## 5. 전략 간 시너지 및 전환

### 전략 전환 시나리오

| 현재 상태 | 이벤트 | 전환 대상 |
|----------|--------|----------|
| 전략 1 실행 중 (추세추종) | 추세 과진행 + 유동성 풀 접근 | → 전략 2 준비 (스윕 헌팅) |
| 전략 2 실행 중 (스윕 헌팅) | 전환 확정 + 새 추세 시작 | → 전략 1로 전환 (눌림목 매매) |
| 전략 3 실행 중 (박스권) | 박스 돌파 + BOS 발생 | → 전략 1로 전환 (돌파 후 눌림목) |
| 전략 1 실행 중 (추세추종) | 추세 약화 + 횡보 시작 | → 전략 3으로 전환 (박스권 매매) |
| 전략 4 시그널 발생 | S-Grade + Smart Signal | → 전략 4 우선 실행 (최고 승률) |

### 전략 동시 운용
- **추세장**: 전략 1 + 전략 4 동시 운용 (전략 4의 엄격한 필터로 고품질 진입만 추가)
- **횡보장**: 전략 3 단독 운용, 돌파 시 전략 1/4로 즉시 전환
- **전환점**: 전략 2 단독, 전환 확정 후 전략 1로 전환
- **불명확**: 전체 보류 → 구조가 명확해질 때까지 대기

## 6. 심리 관리 매트릭스

| 전략 | 주요 심리적 도전 | 대응 방법 |
|------|----------------|----------|
| **전략 1** | "이번 눌림목이 진짜 하락이면?" | OB 이탈 시 무조건 손절. 1번 손절 = 다음 기회 대기 |
| **전략 2** | "7번 연속 손절... 이 전략이 맞나?" | RR이 높으므로 1번 성공이 7번 손절을 보상. 통계를 신뢰 |
| **전략 3** | "매매가 너무 잦아서 지친다" | 1일 최대 매매 횟수 제한 (예: 3회). 무리하면 쉬기 |
| **전략 4** | "시그널이 안 나와서 답답하다" | 엄격한 필터 = 높은 승률. 기다림이 곧 실력 |
| **공통** | "손절 직후 원래 방향으로 갔다" | 모든 전략에서 발생하는 자연현상. 1개 매매가 아닌 100개 매매 관점 |

## 8. AI 오케스트레이션 에이전트 디렉션

### 총괄 에이전트 역할
> 이 총괄 에이전트는 현재 시장 상태를 판단하고, **4가지 전략 전문가 에이전트 중 적합한 에이전트를 호출**하는 오케스트레이터 역할을 한다.

### 오케스트레이션 플로우

```
[입력] 현재 차트 데이터

[Phase 1] 장세 판단
    │
    ├── HTF 시장 구조 분석 (BOS/CHOCH 유무)
    ├── 이평선 배열 상태 (정배열/역배열/수렴)
    ├── 유동성 풀 근접 여부
    ├── 횡보 여부 (스윙 포인트 개수)
    │
    ├── 판정: 추세장 / 전환점 / 횡보장 / 불명확

[Phase 2] 전략 선택 및 전문가 에이전트 호출
    │
    ├── 추세장 → 전략 1 에이전트 + 전략 4 에이전트 동시 호출
    ├── 전환점 → 전략 2 에이전트 호출
    ├── 횡보장 → 전략 3 에이전트 호출
    └── 불명확 → "보류" 판정, 에이전트 미호출

[Phase 3] 전문가 에이전트 결과 수집
    │
    ├── 각 에이전트의 판정(롱/숏/보류) 수집
    ├── 신뢰도, RR, OB 등급 비교
    └── 충돌 시 → 더 높은 신뢰도 + 더 높은 RR 우선

[Phase 4] 최종 통합 판단 출력
```

### 통합 출력 포맷
```
═══════════════════════════════════════
       SMC 통합 매매 판단
═══════════════════════════════════════
[장세] 추세장 (상승) / 전환점 / 횡보 / 불명확
[활성 전략] 전략 1 + 전략 4 (예시)

── 전략 1 (OB 눌림목) ──
[판정] 롱 | [신뢰도] 상 | [OB등급] A급
[진입] xxxx | [손절] xxxx | [익절] xxxx
[RR] 1:2.5 | [사이즈] 1.5%

── 전략 4 (Golden OB) ──
[판정] 롱 | [신뢰도] 상 | [OB등급] S(14점)
[진입] xxxx | [손절] xxxx | [익절] xxxx
[RR] 1:3.0 | [사이즈] 2.0%

── 최종 권고 ──
[채택] 전략 4 (더 높은 등급 + RR)
[포지션] xxxx 롱
[리스크] 계좌의 2%
[주의] 없음
═══════════════════════════════════════
```
