# 전략 개요 (Strategy Overview)

> **핵심 철학**
> 이평선은 후행지표이므로 바닥/천장 예측이 아닌, **추세 확인 후 눌림목 진입**에 최적화된 도구다.
> 쌍바닥/쌍봉으로 추세 전환을 확인한 뒤, 이평선 눌림목에서 진입하여 승률과 손익비를 동시에 확보한다.

## 전략의 전제 조건
- 이평선은 **추세 확인** 도구이지, 예측 도구가 아님
- 횡보장에서는 이평선 전략이 반복적으로 손절에 걸림 → **추세 존재 여부 판단이 선행**되어야 함
- 쌍바닥/쌍봉은 추세 전환의 **확인 신호**이며, 눌림목은 그 추세에 **탑승하는 시점**

---

# 1. 사용 이평선 정의

| 이평선 | 기간 | 역할 |
|--------|------|------|
| 단기 | 5 EMA | 진입/이탈 트리거, 모멘텀 판단 |
| 중기 | 20 EMA | 추세 방향 기준선, 눌림목 지지/저항 |
| 장기 | 60 EMA | 대추세 방향 확인, 배열 판단 기준 |
| 초장기 | 120 EMA | HTF 추세 컨펌 (선택적) |

> **참고**: SMA vs EMA 선택은 자산과 타임프레임에 따라 테스트 필요. 일반적으로 단기 트레이딩에서는 EMA가 반응이 빠르므로 유리.

# 2. 이평선 배열 상태 정의

## 정배열 (Bullish Alignment)
- 5 EMA > 20 EMA > 60 EMA
- 의미: 상승 추세가 확립됨 → **롱 눌림목 진입 환경**

## 역배열 (Bearish Alignment)
- 5 EMA < 20 EMA < 60 EMA
- 의미: 하락 추세가 확립됨 → **숏 눌림목 진입 환경**

## 수렴/교차 (Consolidation/Cross)
- 이평선들이 수렴하거나 꼬여 있는 상태
- 의미: 방향성 미확정 → **매매 보류 구간 (No Trade Zone)**
- ⚠️ 이 구간에서 매매하면 반복적으로 손절됨

# 3. 패턴 정의

## 3-1. 쌍바닥 (Double Bottom) — 롱 전환 신호

### 구조
```
가격
  │
  │    ╲      ╱╲     ╱ ← 넥라인 돌파
  │     ╲    ╱  ╲   ╱
  │      ╲  ╱    ╲ ╱
  │       ╲╱      ╲╱
  │      1차저점  2차저점
  └──────────────────── 시간
```

### 인식 조건
1. **1차 저점**: 하락 추세 중 의미있는 저점 형성
2. **중간 반등**: 1차 저점에서 반등하여 넥라인(중간 고점) 형성
3. **2차 저점**: 1차 저점 부근까지 재차 하락 (1차 저점 대비 ±3% 이내)
4. **넥라인 돌파**: 2차 저점에서 반등하여 중간 고점(넥라인)을 상방 돌파
5. **이평선 확인**: 이 과정에서 가격이 5 EMA를 상방 돌파하고, 5 EMA가 20 EMA를 상향 교차 시작

### 유효성 강화 조건
- 2차 저점이 1차 저점보다 약간 높음 (상승형 쌍바닥) → 더 강한 신호
- 2차 저점에서 거래량 감소 → 매도세 소진 확인
- RSI 등 오실레이터에서 다이버전스 발생 → 추가 확인

## 3-2. 쌍봉 (Double Top) — 숏 전환 신호

### 구조
```
가격
  │       ╱╲      ╱╲
  │      ╱  ╲    ╱  ╲
  │     ╱    ╲  ╱    ╲
  │    ╱      ╲╱      ╲ ← 넥라인 이탈
  │                     ╲
  │      1차고점  2차고점
  └──────────────────── 시간
```

### 인식 조건
1. **1차 고점**: 상승 추세 중 의미있는 고점 형성
2. **중간 조정**: 1차 고점에서 조정하여 넥라인(중간 저점) 형성
3. **2차 고점**: 1차 고점 부근까지 재차 상승 (1차 고점 대비 ±3% 이내)
4. **넥라인 이탈**: 2차 고점에서 하락하여 중간 저점(넥라인)을 하방 이탈
5. **이평선 확인**: 가격이 5 EMA를 하방 이탈하고, 5 EMA가 20 EMA를 하향 교차 시작

### 유효성 강화 조건
- 2차 고점이 1차 고점보다 약간 낮음 (하락형 쌍봉) → 더 강한 신호
- 2차 고점에서 거래량 감소 → 매수세 소진 확인
- RSI 등 오실레이터에서 베어리시 다이버전스 발생

## 3-3. 눌림목 (Pullback) — 추세 탑승 신호

### 롱 눌림목 인식 조건
1. 쌍바닥 이후 가격이 5 EMA 위로 올라탐 (추세 전환 1차 확인)
2. 이평선 정배열 진행 중 (5 > 20, 혹은 정배열 형성 직전)
3. 가격이 조정을 받으며 **5 EMA 또는 20 EMA까지 하락** (이평선에 "붙는" 구간)
4. 이평선 터치 후 **양봉 출현 + 5 EMA 재돌파** → 진입 트리거

### 숏 눌림목 인식 조건
1. 쌍봉 이후 가격이 5 EMA 아래로 내려감 (추세 전환 1차 확인)
2. 이평선 역배열 진행 중 (5 < 20, 혹은 역배열 형성 직전)
3. 가격이 반등하며 **5 EMA 또는 20 EMA까지 상승** (이평선에 "붙는" 구간)
4. 이평선 터치 후 **음봉 출현 + 5 EMA 재이탈** → 진입 트리거

### 눌림목의 질적 평가 (Grade)
| 등급 | 조건 | 신뢰도 |
|------|------|--------|
| A급 | 5 EMA 터치 후 즉각 반등, 거래량 동반 | 최상 |
| B급 | 20 EMA 터치 후 반등, 이평선 배열 유지 | 상 |
| C급 | 60 EMA까지 깊은 조정 후 반등 | 중 (추세 약화 의심) |
| D급 | 60 EMA 이탈 → 추세 훼손 | 매매 불가 |

# 4. 멀티타임프레임 (MTF) 분석 체계

## 4-1. 타임프레임 역할 분담

| 타임프레임 | 역할 | 확인 사항 |
|-----------|------|----------|
| **일봉/4시간봉 (HTF)** | 추세 방향 판단 | 이평선 배열 상태, 큰 구조의 쌍바닥/쌍봉 |
| **1시간봉 (MTF)** | 추세 진행 단계 확인 | 눌림목 발생 구간 식별, 구조 전환 감지 |
| **15분/5분봉 (LTF)** | 정밀 진입 타이밍 | 쌍바닥/쌍봉 트리거, 캔들 패턴 확인 |

## 4-2. Top-Down 분석 플로우

```
[Step 1] HTF(4H) 추세 확인
    │
    ├── 정배열 → 롱 시나리오만 탐색
    ├── 역배열 → 숏 시나리오만 탐색
    └── 수렴/교차 → 매매 보류 (No Trade)
    │
[Step 2] HTF(4H) 눌림목 구간 확인
    │
    ├── 가격이 이평선에 붙은 구간인가?
    ├── 이평선 배열이 유지되고 있는가?
    └── 직전 구조(쌍바닥/쌍봉)가 확인되었는가?
    │
[Step 3] LTF(5m/15m)에서 진입 트리거 탐색
    │
    ├── LTF에서 쌍바닥/쌍봉 패턴 확인
    ├── 5 EMA 돌파/이탈 캔들 확인
    └── 진입 실행
```

# 5. 진입/손절/익절 규칙

## 5-1. 롱 진입

### 진입 조건 체크리스트
- [ ] HTF(4H): 이평선 정배열 또는 정배열 진행 중
- [ ] HTF(4H): 쌍바닥 구조 확인 완료
- [ ] HTF(4H): 가격이 이평선(5 or 20 EMA)에 눌림목으로 접근
- [ ] LTF(5m/15m): 눌림목 구간에서 쌍바닥 패턴 형성
- [ ] LTF(5m/15m): 5 EMA 상방 돌파 양봉 출현
- [ ] 거래량: 돌파 시 거래량 증가 확인 (선택)

### 진입 시점
- LTF에서 5 EMA 상방 돌파 양봉의 **종가 확정 시** 진입
- 또는 돌파 캔들 다음 캔들이 돌파 캔들의 고가를 넘을 때 진입 (보수적)

### 손절
- **기본**: LTF 쌍바닥의 최저점 하방 (2차 저점 기준)
- **보수적**: HTF 눌림목의 최저점 하방
- ⚠️ 직전 저점(IDM)은 유동성 사냥 대상이므로, **쌍바닥의 최저점(절대 저점)** 기준으로 설정
- 손절폭이 너무 넓으면 → 포지션 사이즈를 줄여 리스크 금액을 고정

### 익절
- **1차 익절**: 손절폭의 1:1 지점 (원금 보호)
- **2차 익절**: 손절폭의 1:2 지점
- **3차 익절 / 트레일링**: HTF의 다음 저항 구간 또는 이평선 역배열 전환 시
- 트레일링 스탑: 20 EMA 하방 이탈 시 잔여 포지션 청산

## 5-2. 숏 진입

### 진입 조건 체크리스트
- [ ] HTF(4H): 이평선 역배열 또는 역배열 진행 중
- [ ] HTF(4H): 쌍봉 구조 확인 완료
- [ ] HTF(4H): 가격이 이평선(5 or 20 EMA)에 반등하여 접근 (눌림목)
- [ ] LTF(5m/15m): 눌림목 구간에서 쌍봉 패턴 형성
- [ ] LTF(5m/15m): 5 EMA 하방 이탈 음봉 출현
- [ ] 거래량: 이탈 시 거래량 증가 확인 (선택)

### 진입 시점
- LTF에서 5 EMA 하방 이탈 음봉의 **종가 확정 시** 진입

### 손절
- **기본**: LTF 쌍봉의 최고점 상방
- ⚠️ 직전 고점은 SMC의 IDM(Inducement)으로 유동성 사냥 대상
- → **쌍봉의 절대 최고점 + 약간의 여유(buffer)** 위에 손절 배치
- 손절폭이 너무 넓으면 → 포지션 사이즈를 줄여 리스크 금액을 고정

### 익절
- **1차 익절**: 손절폭의 1:1 지점
- **2차 익절**: 손절폭의 1:2 지점
- **3차 익절 / 트레일링**: HTF의 다음 지지 구간 또는 이평선 정배열 전환 시
- 트레일링 스탑: 20 EMA 상방 돌파 시 잔여 포지션 청산

# 6. 매매 보류 조건 (No Trade Zone)

다음 조건 중 하나라도 해당하면 **매매하지 않음**:

1. **HTF 이평선 수렴/교차 상태** — 방향성 미확정
2. **HTF에서 쌍바닥/쌍봉 구조가 아직 미완성** — 추세 전환 미확인
3. **LTF 눌림목이 C급 이하** — 60 EMA 이하로 깊은 조정
4. **주요 경제 이벤트 직전** — FOMC, CPI, 고용지표 등 (변동성 급등 구간)
5. **손익비 1:1.5 이상 확보 불가** — 손절 대비 기대 수익이 부족
6. **이미 추세가 과도하게 진행된 상태** — 3파 이상 진행 후 눌림목은 실패 확률 높음

# 8. SMC(Smart Money Concept)와의 연계

## IDM (Inducement) 고려사항
- **직전 고점/저점은 유동성 사냥(liquidity grab) 대상**
- 기관(Smart Money)은 개인 트레이더의 손절을 유동성으로 활용
- 따라서 손절을 직전 고/저점에 두면 **털린 후 원래 방향으로 진행**하는 경우가 빈번

### 실전 적용
- 손절은 항상 **구조의 절대 최고점/최저점** 기준
- 직전 고/저점이 깨질 때 바로 손절하지 않고, **쌍봉/쌍바닥의 극단점 이탈** 시 손절
- IDM 스윕 후 원래 방향으로 되돌아오는 것을 기다리는 인내심 필요

## Order Block / FVG 활용 (보조 확인)
- 눌림목 진입 시, 해당 구간이 HTF Order Block과 겹치면 신뢰도 상승
- Fair Value Gap(FVG)이 있는 구간으로의 눌림목은 반등 확률이 높음

# 9. AI 전문가 에이전트 디렉션

## 에이전트 역할 정의
> 이 에이전트는 "이평선 쌍바닥/눌림목 전문가"로서, 주어진 차트 데이터에서 아래 항목들을 순차적으로 분석하고 매매 판단을 제공한다.

## 분석 파이프라인 (순서대로 실행)

### Phase 1: 환경 분석 (HTF)
1. 4H 차트의 이평선 배열 상태 판단 (정배열/역배열/수렴)
2. 쌍바닥 또는 쌍봉 구조 존재 여부 확인
3. 현재 가격과 이평선의 위치 관계 파악
4. → **매매 가능 환경인지 1차 판단** (No Trade Zone 해당 시 중단)

### Phase 2: 눌림목 감지 (HTF → MTF)
5. 가격이 이평선에 접근/터치하는 눌림목 구간 식별
6. 눌림목 등급 판정 (A/B/C/D)
7. D급이면 매매 불가 판정, C급이면 경고와 함께 진행

### Phase 3: 진입 트리거 탐색 (LTF)
8. LTF(5m/15m)에서 쌍바닥/쌍봉 패턴 탐색
9. 5 EMA 돌파/이탈 캔들 확인
10. 진입 가격, 손절 가격, 익절 목표가 산출
11. 손익비 계산 → 1:1.5 미만이면 패스

### Phase 4: 리스크 관리
12. 계좌 대비 리스크 비율 계산 (권장: 계좌의 1~2%)
13. 포지션 사이즈 산출
14. 시나리오 등급에 따른 사이즈 조절 (박스권 시나리오는 50% 축소)

### Phase 5: 최종 판단 출력
```
[판정] 롱/숏/보류
[신뢰도] 상/중/하
[HTF 배열] 정배열/역배열/수렴
[패턴] 쌍바닥 확인 + A급 눌림목 (예시)
[진입가] xxxx
[손절가] xxxx (리스크: x%)
[익절 1차] xxxx (R:R 1:1)
[익절 2차] xxxx (R:R 1:2)
[포지션 사이즈] 계좌의 x%
[주의사항] (해당 시)
```

# 10. 전략 한계 및 주의사항

1. **횡보장 취약**: 이평선이 수렴하면 시그널이 무의미해짐 → No Trade Zone 엄수
2. **후행성**: 이평선 자체가 후행지표이므로, 급격한 추세 전환 시 늦은 진입 가능
3. **쌍바닥/쌍봉 오인식**: 비슷하지만 유효하지 않은 패턴 존재 → 넥라인 돌파/이탈 확정 전까지는 확인 불가
4. **과최적화 주의**: 백테스트에서 이평선 기간을 자산별로 최적화하되, 과적합에 주의
5. **뉴스/이벤트 리스크**: 기술적 분석만으로는 대응 불가한 외부 변수 존재

# 부록: 용어 정리

| 용어 | 설명 |
|------|------|
| EMA | Exponential Moving Average (지수이동평균) |
| 정배열 | 단기 > 중기 > 장기 이평선 순서 (상승 추세) |
| 역배열 | 단기 < 중기 < 장기 이평선 순서 (하락 추세) |
| 눌림목 | 추세 방향으로 진행 중 일시적 되돌림 (조정) |
| 넥라인 | 쌍바닥/쌍봉의 중간 고점/저점 수평선 |
| MTF | Multi Time Frame (멀티타임프레임) |
| HTF | Higher Time Frame (상위 타임프레임) |
| LTF | Lower Time Frame (하위 타임프레임) |
| IDM | Inducement — SMC 개념으로, 유동성 사냥을 위한 미끼 구간 |
| FVG | Fair Value Gap — 급격한 가격 이동으로 생긴 캔들 간 갭 |
| R:R | Risk to Reward Ratio (손익비) |
